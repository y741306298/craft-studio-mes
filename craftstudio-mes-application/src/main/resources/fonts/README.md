# Fonts for tag rendering

Please place the open-source font file below into this directory:

- File name: `SourceHanSansSC-Regular.otf`
- Font family used by code: `Source Han Sans SC`
- Upstream project: Adobe Source Han Sans (思源黑体)
- Suggested download URL:
  - https://github.com/adobe-fonts/source-han-sans/raw/release/OTF/SimplifiedChinese/SourceHanSansSC-Regular.otf

After copying the file, ensure runtime environment can discover this font (or install system-wide).
